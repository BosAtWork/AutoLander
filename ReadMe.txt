Requirements:
 Compile
  - FlashDevelop 4+ (tested only for 4+)
  - Flex SDK
  - Java SDK
  - Microsoft .NET Runtime 2 (for running flashdevelop)
 Run
  - Flash player 10+
  - Browser or stand alone player
  - High score needs cookies enabled

Notes:
  - The editor is easy to break. Use at own risk.
  - The editor generates class files you need to copy and create in src/data
  - In Config.as you can add the level in the Config.LEVELS array.
  - To view the debug variables set the Config.DEBUG on true instead of false.
